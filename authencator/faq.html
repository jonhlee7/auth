How Authenticator Works
What is an Authenticator? +
An Authenticator is a security application used to generate one-time passwords (OTPs) or codes. These codes provide an additional layer of security for various accounts, often as a part of two-factor authentication (2FA) processes.
How does it generate codes? +
The Authenticator app uses a time-based or counter-based algorithm to generate a unique, temporary code. For time-based tokens, each code is valid for a short period (usually 30 seconds) before a new one is generated.
Do I need an internet connection to use Authenticator? +
No, internet access is not required to generate codes. The app functions offline as it relies on a pre-shared key and the current time or a counter to generate the OTP.
How do I set up Authenticator with my accounts? +
To link Authenticator with an account, you typically scan a QR code provided by the service you're securing, which syncs the account with your Authenticator app. Alternatively, you can enter a setup key manually.
Is Authenticator secure? +
Yes, Authenticator apps are considered highly secure. Since the codes are generated on your device and change frequently, they are difficult for attackers to compromise. However, physical security of the device hosting the Authenticator is also crucial.
General Issues
What should I do if I fail to add an account by scanning the QR code? +
Ensure that your camera permission is enabled and the QR code is clear and well-lit. If the issue persists, you can try to add the account manually by entering the details.
Why can't I use Face ID or Touch ID for verification? +
Please verify that your device supports Face ID or Touch ID and that it is activated in your system settings. If you still face issues after correct setup, try restarting your device or reconfiguring Face ID or Touch ID in the settings.
Will I lose my existing account information if I switch phones? +
Not if you have previously backed up your accounts. You can back up via iCloud or other supported cloud services and restore on your new device.
Does the Authenticator support syncing with Apple Watch? +
Yes. Ensure that your iPhone is properly paired with your Apple Watch and that the Apple Watch syncing feature is enabled within the Authenticator app.
Where is my backup stored? How can I restore from a backup? +
Your backups are stored in your chosen cloud service, like iCloud. In the "Backup & Restore" settings, you can find options to restore from the selected backup file.
What should I do if my one-time password does not work? +
Ensure that your device's time is accurately set and is configured to automatically synchronize. Discrepancies in time can render one-time passwords invalid.
Can I recover an account if I accidentally deleted it? +
You can restore it from "Recently Deleted" if it was removed recently. If it has been cleared from there, you would need to restore from a previous backup.
How should I update the Authenticator app? +
To ensure security and optimal performance, enable automatic app updates or regularly check for updates manually in the app store.
What should I do if I encounter issues while using the Authenticator? +
If you encounter any problems, please contact our customer service team through the "Help" or "Contact Us" section of the app, and we will provide the necessary support.
How can I make my accounts more secure? +
In addition to enabling two-step verification with the Authenticator, we recommend changing passwords regularly, using strong passwords, and avoiding the use of the same password across different services.
Backup and Restore
How do I set up automatic backups? +
In the "Backup & Restore" settings, activate the auto-backup option, set the start time, and the frequency of backups. You can choose to back up daily or weekly to ensure your data is always up to date.
What should I pay attention to during backup and restoration? +
Ensure that your network connection is stable when backing up or restoring, and you are using the latest version of the Authenticator app. Also, the account used for backup should be the same as the one used for restoration.
